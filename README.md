# SteamDownloader

Download or search any items/collections.

## Dependencies

- You need [SteamCmd](https://developer.valvesoftware.com/wiki/SteamCMD#Downloading_SteamCMD) installed on your pc and specify it's location in `config.json`.

### How to use

- Clone this repo
- Tweak `config.json`
- `python main.py`
